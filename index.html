Hexo

工具

安装

依赖

node
yarn
yarn global add hexo

start

mkdir blog && cd blog
yarn init

github pages

add repo

仓库名必须是github用户名，比如username.github.io

config repo

settings -> GitHub Pages -> 选择master分支 -> save

deploy

安装插件

yarn add hexo-deployer-git

修改配置文件

deploy:
type: git
repo: git@github.com:username/username.github.io.git
branch: master

配置

主题

下载后，修改根配置下的主题名称；
主题配置中修改相关项。

写作

添加page
添加文章（分类、标签）

配置后的动作

  • hexo clean

  • hexo generate

  • hexo deploy

访问

https://yourgithubname.github.io
